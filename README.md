## 📘 About
This repository contains the official **Master Lab Manual** and **Solved Lab Reports** for the course **Digital System Design (CPE344)**.
The lab focuses on:
- Hardware Description Language (**Verilog HDL**)
- Digital system modeling, simulation, and verification
- Finite State Machines (FSMs)
- FPGA implementation using Spartan-6
- Design of combinational and sequential circuits

## 📋 List of Labs
| Lab No. | Topic |
|---------|-------|
| **Lab 01** | Introduction to Xilinx Software, Verilog modeling of logic gates, Half Adder & Full Adder (Gate Level Modeling) |
| **Lab 02** | Transmission Gate Primitives, Half Subtractor & Full Subtractor (Gate Level & Dataflow Modeling) |
| **Lab 03** | Introduction to FPGA and Implementation of Logic Gates & Combinational Circuits on Spartan-6 |
| **Lab 04** | Physical Implementation of Ripple Carry Adder, Carry Lookahead Adder & Multiplexers on FPGA |
| **Lab 05** | Implementation of Combinational Circuits using Behavioral Modeling (Full Adder, Subtractor, MUX, Multiplier) |
| **Lab 07** | Implementation of Encoders using Conditional Operators on FPGA |
| **Lab 08** | Decoders and Comparators using Behavioral Modeling |
| **Lab 09** | Implementation of Ripple Carry Adder and Carry Lookahead Adder |
| **Lab 10** | Design and Implementation of Multipliers and Arithmetic Logic Unit (ALU) |
| **Lab 12** | Advanced Delay Models in Verilog + Modeling of Latches and Flip-flops |
| **Lab 13** | 4-bit Shift Register and Majority Counter |
| **Lab 14** | Finite State Machines (Sequence Detector & Edge Detector using Moore and Mealy models) |
